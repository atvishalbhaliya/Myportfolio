# Portfolio (Static)

Static portfolio website built with plain HTML/CSS and a tiny bit of JavaScript.

## Edit your details

Update:

- `index.html` (GitHub/LinkedIn links, projects if you want to add more)
- `assets/avatar-placeholder.svg` (initials)

## Preview locally

Option 1 (no tools): open `index.html` in a browser.

Option 2 (quick open via PowerShell):

```powershell
Start-Process .\\index.html
```

Option 3 (recommended, live reload): run a local server (any one you have installed), e.g.:

```powershell
python -m http.server 5500
```

Then open `http://localhost:5500`.
